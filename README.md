# IMU
该程序利用本机蓝牙通过搜寻蓝牙名字连接外设蓝牙，连接成功后会订阅其通知，接收外设蓝牙数据并绘制出空间姿态和三轴数据曲线。

# uart
该程序用串口绘制曲线图

# can
该程序利用周立功can通讯器件进行二次开发库上，对can数据进行曲线绘制。

# keil
该程序用keil导出的hex文件直接绘制曲线图
